# Browser Extension Security Review

## Overview
This task focuses on reviewing installed browser extensions to identify potentially unnecessary or risky add-ons and improve browser security.

A manual inspection was performed on all installed extensions, including checking their purpose, permissions, and necessity. Based on the analysis, one extension was removed to reduce potential security risks.

---

3. **Photoshop Online**

   - Status: Removed
   - Reason: Duplicate functionality and unnecessary third-party extension

---

## Action Taken

- Removed **Photoshop Online** extension to reduce browser exposure and minimize unnecessary permissions.
- Retained only essential and trusted extensions.

---

## Outcome

- Browser security posture improved by limiting installed extensions.
- Reduced attack surface by removing redundant third-party add-ons.
- Improved awareness of browser extension risks and management best practices.
